视频混剪合并工具技术方案

一、目标
- 输入前半段与后半段两组视频素材
- 两两组合生成新视频
- 跨平台运行，Mac 与 Windows 均可用
- 处理分辨率不一致，输出统一尺寸

二、技术选型
- 语言：Python 3
- UI：Tkinter（内置，无额外依赖）
- 合并引擎：FFmpeg + FFprobe
- 打包：PyInstaller（生成 .app / .exe）

三、核心流程
1) 扫描前半段目录与后半段目录，筛选视频文件
2) 计算组合任务数量（前段数量 × 后段数量）
3) 对每一对视频执行合并
   - 优先无损：使用 concat demuxer + copy
   - 无损失败或强制兼容：重编码 + 统一分辨率
4) 输出到指定目录并记录日志

四、统一分辨率策略
- 自动模式：以前半段分辨率为目标尺寸
- 自定义模式：用户手动输入目标宽高
- 兼容模式处理方式
  - 使用 scale + pad 保持比例，避免拉伸
  - 输出统一画布大小

五、音频策略
- 前后段均有音频：保留音频并拼接
- 任一段无音频：输出无音频视频

六、目录结构建议
- main.py
- ffmpeg/ffmpeg(.exe) 与 ffprobe(.exe)
- 使用说明.md
- 技术方案.md

七、打包方案
- 安装 PyInstaller
  - pip install pyinstaller
- macOS 打包
  - pyinstaller --onefile --add-binary "ffmpeg;ffmpeg" --add-binary "ffprobe;ffmpeg" main.py
- Windows 打包
  - pyinstaller --onefile --add-binary "ffmpeg.exe;ffmpeg" --add-binary "ffprobe.exe;ffmpeg" main.py

八、可扩展方向
- 组合数量限制与随机组合
- 任务暂停与断点续跑
- 合并失败自动重试与错误报告导出
